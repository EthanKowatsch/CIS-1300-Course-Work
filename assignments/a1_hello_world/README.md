# Assignment 1 — Hello World & Intro to Linux

## Overview
This assignment introduced the basics of working in the SoCS Linux environment and building simple C programs using the standard edit–compile–run workflow. It also provided early experience using AI tools as code reviewers in a responsible and critical way.

## Topics List
- Variables
- Conditionals
- Loops
- Functions
- Input/Output
- Basic error handling

## How to Compile
Use the following command to compile the initial program:

```bash
gcc -Wall -std=c99 a1_initial.c -o initial
./initial
```
Use the following command to compile the final program:

```bash
gcc -Wall -std=c99 a1_final.c -o final
./final
```
